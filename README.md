Calories Burn Prediction using Machine Learning
Overview

A machine learning regression project that predicts calories burned based on user input features.

Tech Stack
Python
Flask
Scikit-learn
Pandas
NumPy
HTML, CSS
Chart.js
Model Used
Random Forest Regressor (Best Model)
How to Run
pip install -r requirements.txt
python app.py
Input Features
Gender
Age
Height
Weight
Duration
Heart Rate
Body Temperature
Output

Calories Burned (Prediction)

Note
Regression problem
No classification metrics used
