Calories Burn Prediction using Machine Learning

Overview

A machine learning regression project that predicts calories burned based on user input features.

Tech Stack:
1) Python
2) Flask
3) Scikit-learn
4) Pandas
5) NumPy
6) HTML
7) CSS
8) Chart.js
9) Model Used
10) Random Forest Regressor (Best Model)

How to Run
1) pip install -r requirements.txt
2) python app.py

Input Features
1) Gender
2) Age
3) Height
4) Weight
5) Duration
6) Heart Rate
7) Body Temperature

Output
Calories Burned (Prediction)

Note
Regression problem
No classification metrics used
